Adds embeded video support to the CLUEVO LMS. Embed YouTube, Vimeo, Soundcloud content and more.

== Description ==
This extension to the CLUEVO LMS enables you to add embeded videos from YouTube, Vimeo and lots of other sites to your learning management system

== Installation ==

1. Upload the plugin files to your plugins directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add your videos and other content through the module management page
